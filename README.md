# WXT-1stExtension

WXTで拡張機能を作る

---

# 手順

## Nodeを最新化

```sh
volta install node@latest
volta install npm@latest
```

## WXTのインストール

```sh
npx wxt@latest init 1stext
```

> ✔ Choose a template › vue
>
> ✔ Package Manager › npm

```sh
cd 1stext

# volta pin node@23.5.0
# volta pin npm@11.0.0

npm install
npm run dev
```
